# Welcome to Rizztaurant 🍱🍝

## Get started

1. Clone the rep
   ```bash
   git clone https://github.com/shijiken/rizztaurant.git
   cd rizztaurant
   ```
  
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npx expo start
   ```

Then open the app in a [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
